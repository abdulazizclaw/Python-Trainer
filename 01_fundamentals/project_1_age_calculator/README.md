# Project 1: Age Calculator

## Objective
Build a program that calculates your age and provides interesting facts about it.

## What You'll Practice
- Taking user input with input()
- Converting between data types (string to int)
- Using variables to store data
- Doing math with numbers
- Printing formatted output

## Project Requirements

Create a program that:

1. Asks the user for their birth year
2. Asks the user for the current year
3. Calculates their age
4. Tells them:
   - How old they are
   - How old they'll be next year
   - How old they were 5 years ago
   - How many months old they are (approximately)

## Example Output

```
Welcome to the Age Calculator!
What year were you born? 1990
What is the current year? 2024
================================
Your age: 34 years old
Next year: 35 years old
5 years ago: 29 years old
Approximately 408 months old
================================
```

## Tips

- Use `input()` to get data from the user
- Use `int()` to convert the user's text input to a number

- Use arithmetic to calculate ages
- Use `print()` to display results
